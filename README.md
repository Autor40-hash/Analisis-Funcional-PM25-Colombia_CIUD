#Análisis Funcional de PM2.5 en Colombia

Este repositorio contiene los datos y códigos utilizados en el artículo sobre el análisis de concentraciones de PM2.5 en Bogotá, Medellín y Cali mediante técnicas de análisis de datos funcionales.

## Contenido

- Bases de datos utilizadas en el estudio.
- Scripts en R para el procesamiento y análisis de los datos.
- Código para la generación de tablas y figuras.

## Reproducibilidad

Los archivos incluidos permiten reproducir los principales resultados presentados en el artículo.
